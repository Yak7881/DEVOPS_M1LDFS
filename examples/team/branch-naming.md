# Nommage des branches

Convention pour nommer tes branches de manière claire.

---

## Format

```
<type>/<description-courte>
```

---

## Types courants

| Préfixe | Usage |
|---------|-------|
| `fix/` | Correction de bug |
| `feature/` | Nouvelle fonctionnalité |
| `chore/` | Tâche de maintenance (dépendances, config) |
| `docs/` | Modification de la documentation |

---

## Exemples

| Contexte | Bon nom |
|----------|---------|
| Bug sur le bouton de connexion | `fix/bouton-connexion-desactive` |
| Nouvelle page "À propos" | `feature/page-a-propos` |
| Mise à jour des dépendances | `chore/update-deps` |
| Correction de typo dans le README | `docs/fix-typo-readme` |

---

## Règles

- Tout en minuscules
- Tirets entre les mots (pas d'espaces)
- Court et descriptif
