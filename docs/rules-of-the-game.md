# Règles du jeu

Ce parcours est un jeu. Voici les règles.

---

## Objectif

Traverser toutes les rooms, faire les missions, passer les checkpoints, et accumuler des points pour obtenir un badge.

---

## Les règles

### 1. Suis l'ordre des rooms
Les rooms sont numérotées de 00 à 11. Fais-les dans l'ordre.  
Chaque room prépare la suivante.

### 2. Lis d'abord, fais ensuite
Dans chaque room :
1. Lis le **README.md** (la leçon)
2. Fais les **missions** (missions.md)
3. Passe le **checkpoint** (checkpoint.md)

### 3. Pas de triche (mais de l'entraide, oui)
- Ne regarde pas les réponses du checkpoint avant d'avoir essayé.
- Tu peux chercher sur le web, c'est normal.
- Tu peux demander de l'aide à quelqu'un.

### 4. Note ta progression
Ouvre [PROGRESS.md](../PROGRESS.md) et coche les étapes terminées.  
Note ton score à chaque checkpoint.

### 5. Pas de stress
- Pas de limite de temps.
- Tu peux refaire une room autant de fois que tu veux.
- L'important, c'est de comprendre, pas d'aller vite.

---

## Système de points

Chaque room rapporte des points.  
Détail complet dans [scoring.md](scoring.md).

| Action | Points |
|--------|--------|
| Lire le README | 2 |
| Terminer une mission | 3 |
| Checkpoint : quiz correct | 1 par bonne réponse |
| Checkpoint : tâche faite | 5 |

---

## Badges

Ton score total détermine ton badge. Voir [scoring.md](scoring.md).
